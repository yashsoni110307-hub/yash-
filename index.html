Under the hood

How does computer vision work?

At a high level, computer vision systems convert images into numbers, extract patterns, and map those patterns to useful outputs such as labels, boxes, or masks.

01

Capture & preprocess the image

Read image pixels (RGB values), resize to a fixed size, normalize intensity, and sometimes convert to grayscale. This makes the data consistent for the model.

Code: OpenCV or Pillow in Python

02

Extract features

Classical CV uses filters (edges, corners, texture) and descriptors (SIFT, HOG). Modern deep learning lets convolutional neural networks learn features automatically.

Modern approach: Convolutional Neural Networks (CNNs)

03

Predict & post-process

The model predicts outputs (class labels, object boxes, masks). Then we filter predictions (like non-max suppression) and visualize the result on top of the original image.

Output: labels, boxes, keypoints, masks, depth, etc.

Core building blocks

Key computer vision tasks

Most projects you build will be a combination of a few standard tasks. Start by understanding these building blocks.

1
Image classification ‚Äì given an image, predict a single label (e.g., "cat" vs "dog"). Good for simple recognition tasks.
2
Object detection ‚Äì find and classify multiple objects with bounding boxes (e.g., people and cars in a street scene).
3
Segmentation ‚Äì label each pixel to know which object it belongs to. Useful for medical images and autonomous driving.
4
Keypoint detection ‚Äì locate specific points like facial landmarks or human joints for pose estimation.
5
Tracking ‚Äì follow objects across frames in a video, giving them consistent IDs over time.
Concept: from pixels to predictions 
# An image is just a grid of numbers
# Example using PyTorch

import torch

# Fake batch of 8 RGB images, 3x224x224 pixels each
images = torch.randn(8, 3, 224, 224)

# A tiny "model": flatten and apply a linear layer
model = torch.nn.Sequential(
    torch.nn.Flatten(),             # [8, 3*224*224]
    torch.nn.Linear(3*224*224, 10)  # 10 classes
)

logits = model(images)  # shape: [8, 10]
probs = torch.softmax(logits, dim=1)

print("Predicted class for first image:", probs[0].argmax().item())

Real models use convolutional layers and train on thousands of images, but the idea is the same: images ‚Üí numbers ‚Üí learn patterns ‚Üí predictions.

Hands-on

Starter code: load and display an image

Here is a minimal Python example using OpenCV to load, convert, and display an image. Try to run and modify it on your own machine.

Python ¬∑ OpenCV 
import cv2

# 1. Load an image from file
img = cv2.imread("image.jpg")  # put your image path here

if img is None:
    raise FileNotFoundError("Could not load image. Check the path.")

# 2. Show original image
cv2.imshow("Original", img)

# 3. Convert to grayscale
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
cv2.imshow("Grayscale", gray)

# 4. Detect edges (Canny)
edges = cv2.Canny(gray, 100, 200)
cv2.imshow("Edges", edges)

# 5. Wait for a key press and close windows
cv2.waitKey(0)
cv2.destroyAllWindows()

Install OpenCV with pip install opencv-python. Then replace image.jpg with any picture you like and experiment with the edge thresholds.

Roadmap for you

Suggested learning path

Since you are a student focusing on coding and research, follow this rough sequence. At each stage, build at least one small project.

Step 1 ¬∑ Foundations 2‚Äì3 weeks
  • Python basics (functions, loops, NumPy arrays).
  • Image as arrays (height √ó width √ó channels).
  • Use OpenCV/Pillow to crop, resize, rotate images.
Step 2 ¬∑ Classical CV 3‚Äì4 weeks

  • Try reproducing a simple research paper result.
